## Problem Statement

> *"How might we transform the hospital waiting experience into a calm, informed, and inclusive journey — so that patients of all ages, abilities, and backgrounds feel supported from the moment they arrive, rather than anxious and forgotten?"*

---

## Design Thinking Process

### Stage 1 — Empathize & Define

- Conducted research using peer-reviewed studies (PMC, ScienceDirect, Frontiers in Communication)
- Created **10 detailed personas** covering elderly, children, chronic illness, first-timers, pregnant women, caregivers, professionals, non-English speakers, wheelchair users, and dementia patients
- Built **10 empathy maps** (Thinks / Feels / Says / Does) for each persona
- Identified core pain points: uncertainty, sensory overload, accessibility gaps, language barriers

### Stage 2 — Ideate

Brainstormed features using the HMW framework, MindMeister mind maps, and Miro sticky note boards.

#### 5 Compulsory Features

| # | Feature | Purpose |
|---|---------|---------|
| 1 | Queue Progress Tracker | Real-time position + estimated wait time |
| 2 | Distraction Hub | VR nature walks, games, calming videos |
| 3 | Silent Pager Notifications | Vibration alert when turn is near |
| 4 | Mood-Based Interactions | Empathy messages + breathing exercises |
| 5 | Feedback Section | Stress rating + suggestions |

#### Additional Innovative Features

- 🌐 Multilingual Support (8 regional languages)
- 👥 Caregiver Companion Mode
- ♿ Accessibility Mode (large text, high contrast, mobility flag)

📁 [`/Docs/Problem_Statement_Ideation.docx`](./Docs/Problem_Statement_Ideation.docx)

### Stage 3 — Prototype

Built a fully interactive HTML prototype simulating all 5 compulsory features plus accessibility and multilingual screens.

#### Screens Included
- 🏠 Home / Queue Dashboard
- 🎮 Distraction Hub (with live breathing exercise)
- 🔔 Silent Alerts & Notification Preferences
- 💙 Mood Check-In with personalised responses
- ⭐ Feedback Section (star rating + stress slider)
- ♿ Accessibility Settings

### Stage 4 — Test

- Tested with **5 users** representing different persona types
- Collected feedback via rating scales and verbal notes
- Average stress level dropped from **7.4/10 → 4.2/10** (43% reduction)
- Iterated prototype based on feedback — 5 documented changes
